# How To Use

## 请求
1.请求API的地址`<host_name>`为`http://api.jiushouguoji.hk`  
2.每个API请求要带上`session`,`channel`字段:
```php  
  $field['session'] = session_id();
  $field['channel'] = 'wap';
```
3.使用`POST`方式请求

> 此API文档仅展示 请求需要的参数 和 响应结果字段
