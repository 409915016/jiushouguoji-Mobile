基于 ThinkPHP5 模板渲染的微信端购物商城：
- 支持微信支付
- 基于 `Gulp` 构建
- 使用 `Less` 管理多页面样式
- 使用淘宝多终端适配方案 `flexible.js`
- 多处使用 `CSS3 Flexible Box` 布局方式

<h3 align="center">开始</h3>

```bash
yarn install 
```
then
```bash
gulp 
```
